# 最大利益アルゴリズムの説明

## コード
```javascript
dfmx = Math.max(R[i] - mn, dfmx);
```

## 処理の説明
ここでやっているのは「今の値 R[i] で売ったら、これまでの最小値で買った時と比べてどれくらい得するか？」を計算して、その差が過去の最大値より大きければ更新する、という処理です。

## 変数の意味
- `R[i] - mn` … 現在値とこれまでの最小値の差
- `dfmx` … これまで見つかった最大の差
- `Math.max()` … どちらが大きいかを選ぶ

## 実行例
たとえば途中経過がこうなります：

| i | R[i] | mn | R[i] - mn | dfmx(更新後) |
|---|------|----|-----------|--------------|
| 0 | 5    | 10^10 | -9999999995 | -9999999995 |
| 1 | 3    | 5      | -2          | -2          |
| 2 | 1    | 3      | -2          | -2          |
| 3 | 3    | 1      | 2           | 2           |
| 4 | 4    | 1      | 3           | 3           |
| 5 | 3    | 1      | 2           | 3           |
